10.1: Pagar Excursio: Pagar amb compte Bancari

Criteris d'acceptacio:

  • A -> el soci paga amb exit una excuriso que esta pendent de pagament
  • B -> el soci vol pagar una excursió que ja ha estat pagada anteriorment
  • C -> el soci vol pagar una excursio pero no introdueix be les dades del pagament en qüestió
  • D -> el soci vol pagar una excursio amb compte bancari però no te el seu compte afegit i per tant no pot fer el pagament
